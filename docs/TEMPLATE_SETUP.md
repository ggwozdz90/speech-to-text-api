# FastAPI Project Template

## Overview

The repository is configured to automatically synchronize with the template repository using the `template-sync.yml` workflow. This workflow runs on a schedule and can also be triggered manually. It will create a pull request with the latest changes from the template repository.

## Template Repository

The template repository used for this project can be found at [ggwozdz90/fastapi-project-template](https://github.com/ggwozdz90/fastapi-project-template).
